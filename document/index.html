Quick Start

Start analyzing your data in just a few lines

Basic Usage 
import pandas as pd
from topanda import TopologicalAnalyzer

# Load your data
df = pd.read_csv("your_data.csv")

# Initialize analyzer
analyzer = TopologicalAnalyzer(df)

# Compute persistent homology
persistence = analyzer.compute_persistence()

# Visualize results
analyzer.plot_persistence_diagram()

Examples

See topanda in action

Persistence Diagram Example 
from topanda import TopologicalAnalyzer
import pandas as pd

# Create sample data
df = pd.DataFrame({
    "x": [1, 2, 3, 4, 5],
    "y": [2, 4, 1, 5, 3]
})

analyzer = TopologicalAnalyzer(df)
diagram = analyzer.compute_persistence_diagram(max_dimension=1)
analyzer.visualize_persistence()
Mapper Algorithm Example 
from topanda import Mapper
import pandas as pd

df = pd.read_csv("data.csv")
mapper = Mapper(df)

# Apply mapper with custom parameters
graph = mapper.fit_transform(
    n_intervals=10,
    overlap=0.3,
    clusterer="dbscan"
)

mapper.visualize_graph()
Topological Clustering Example 
from topanda import TopologicalClustering
import pandas as pd

df = pd.read_csv("data.csv")
clusterer = TopologicalClustering(df)

# Perform topological clustering
labels = clusterer.fit_predict(
    n_clusters=5,
    metric="euclidean"
)

print(f"Found {len(set(labels))} clusters")
